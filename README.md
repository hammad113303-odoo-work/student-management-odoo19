> # 🎓 Student Management System (Odoo 19)

## 📖 Overview

This is a Student Management System developed in Odoo 19 for learning Odoo Functional and Technical Development.

The project demonstrates how to build a complete business application using Odoo models, views, menus, security, sequences, reports, and relationships.

---

## ✨ Features

- Student Management
- Teacher Management
- Guardian Management
- Class Management
- Subject Management
- Student Reports
- Automatic Sequences
- Access Rights
- Smart Buttons
- PDF Reports
